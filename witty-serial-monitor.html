Witty シリアルモニター

Witty Timer が USB 経由でどんなデータを送ってくるかを確認する診断ツール

このブラウザは Web Serial API に対応していません。Chrome または Edge で http://localhost:5173/witty-serial-monitor.html を開いてください。

使い方

  1. Witty Manager を完全に終了する(ポートを取り合うため必須)
  2. Witty Timer を USB ケーブルで PC に接続し、電源を入れる
  3. 下の「ポートを選んで接続」を押し、一覧に出るポートを選ぶ(出ない場合はボーレートを変えて再試行)
  4. 接続したまま Witty で計測を実行し、スタート/ゴールの瞬間にログへ何か流れるかを観察する
  5. 何か流れたら「ログを保存」で記録して共有してください。プロトコル解析に進めます
