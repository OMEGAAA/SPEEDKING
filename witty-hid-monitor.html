Witty HIDモニター

Microgate Witty 系デバイス (VID 0x16D0, HIDデバイス) が送ってくるレポートを観察する診断ツール

このブラウザは WebHID API に対応していません。Chrome または Edge で http://localhost:5173/witty-hid-monitor.html を開いてください。

使い方

  1. Witty Manager を完全に終了する(デバイスを取り合うため)
  2. Witty Timer を USB 接続して電源を入れ、「Witty に接続」を押す(一覧に Witty が出るので選択)
  3. 接続直後に出る「デバイス構成」のログ(レポートID・サイズの一覧)がまず重要な情報です
  4. 接続したまま Witty で計測を実行し、スタート/ゴールの瞬間に受信レポートが流れるかを観察する
  5. 結果に関わらず「ログを保存」で記録して共有してください
